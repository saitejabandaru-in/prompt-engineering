## Core Frameworks

### COSTAR Framework
- **Context:** Background the AI needs
- **Objective:** What exactly you want
- **Style:** How you want it written
- **Tone:** Formal, casual, technical?
- **Audience:** Who reads this?
- **Response:** Format, length, structure

### Chain-of-Thought
Ask the model to reason step by step before answering.
Improves accuracy on complex tasks by 30-50%.

### Few-Shot Prompting
Show examples of input/output pairs before your actual task.

## Power Prompts

### Data Analysis
You are a Senior Data Analyst. Analyze this dataset and provide:
1. Key business insights (top 3)
2. Anomalies worth investigating
3. Recommended next steps
4. 3 SQL queries to validate your findings

### Code Review
You are a Senior Python Developer. Review this code for:
bugs, performance issues, security vulnerabilities, and code quality.
For each issue: line number, problem, fix, explanation.
